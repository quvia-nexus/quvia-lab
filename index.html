Brand Story

The Way from Question to Value

"Q(Question) + via(라틴어: ~을 통해, 길)
질문을 통해 나아가는 탐구의 여정"
QUVIA = QUESTION + VIA

큐비아 탐구학원은 학생들이 단순히 지식을 배우는 것을 넘어 스스로 질문하고, 탐구하고, 실험하고, 연구하고, 결과를 만들어내는 교육을 지향합니다.

Q
Question
모든 탐구의 시작점. 신중하게 질문하라. 좋은 질문이 좋은 탐구를 만든다.
E
Explore
문제를 파고드는 과정. 깊게 탐구하라. 표면이 아닌 본질을 조사한다.
P
Prove
결과물로 실력을 입증하라. 대담하게 증명하라. 탐구는 결과물로 완성된다.

LV.1
Sense
감지 단계
유아~초등 1-2학년

교과 연계 스토리텔링으로 코딩을 처음 만나는 단계. ScratchJr를 활용해 수학·국어·과학 교과 주제를 직접 움직이는 작품으로 표현하고, 로봇 코딩으로 명령어가 현실에서 작동하는 원리를 몸으로 익힙니다.

LV.2
Spark
점화 단계
초등 2-4학년

엔트리로 나만의 첫 번째 프로그램을 완성하는 단계. 변수·조건·반복의 핵심 개념을 배우고, 로봇 코딩으로 알고리즘을 구현하며, 메타버스 환경에서 디지털 공간 설계까지 경험합니다.

LV.3
Probe
탐구 단계
초등 4-6학년

디지털과 물리 세계를 연결하는 피지컬 컴퓨팅 단계. 엔트리 기반 응용 프로그램 개발과 함께, 센서·모터를 활용한 피지컬 코딩, 앱인벤터로 나만의 스마트폰 앱을 직접 제작합니다.

LV.4
Build
제작 단계
초등 6학년~중등 1학년

AI가 어떻게 학습하는지 원리부터 이해하는 단계. 스크래치 기반 머신러닝 도구로 인공지능 모델을 직접 훈련시키고, 피지컬 디바이스와 융합해 AI가 실제로 작동하는 결과물을 제작합니다.

LV.5
Master
통달 단계
중등~

실전 언어로 문제를 해결하는 최종 단계. Python을 활용한 교과 융합 프로젝트, 정렬·탐색 알고리즘 설계, 데이터 분석 및 인공지능 모델 구현까지 — 소프트웨어 역량을 종합적으로 완성합니다.
