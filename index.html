Brand Story

The Way from Question to Value

"Q(Question) + via(라틴어: ~을 통해, 길)
질문을 통해 나아가는 탐구의 여정"
QUVIA = QUESTION + VIA

큐비아 탐구학원은 학생들이 단순히 지식을 배우는 것을 넘어 스스로 질문하고, 탐구하고, 실험하고, 연구하고, 결과를 만들어내는 교육을 지향합니다.

Q
Question
모든 탐구의 시작점. 신중하게 질문하라. 좋은 질문이 좋은 탐구를 만든다.
E
Explore
문제를 파고드는 과정. 깊게 탐구하라. 표면이 아닌 본질을 조사한다.
P
Prove
결과물로 실력을 입증하라. 대담하게 증명하라. 탐구는 결과물로 완성된다.

LV.1
Sense
감지 단계
유아~초등 1-2학년

코딩과의 첫 만남을 놀이와 탐구로 설계한 입문 단계입니다. 누리과정·교과 주제와 연계된 스토리텔링 기반 ScratchJr 프로젝트를 통해 컴퓨팅 사고력의 기초를 형성하고, 로봇 코딩으로 명령의 순서와 흐름을 익힙니다.

LV.2
Spark
점화 단계
초등 2-4학년

알고리즘적 사고를 실제 프로그램으로 구현하는 단계입니다. 엔트리 기반 프로젝트에서 변수·조건·반복 구조를 체득하고, 로봇 코딩으로 알고리즘의 물리적 작동을 검증하며, 메타버스 플랫폼에서 학생 주도의 디지털 공간을 직접 설계·발표합니다.

LV.3
Probe
탐구 단계
초등 4-6학년

소프트웨어와 하드웨어의 융합 설계를 경험하는 탐구 단계입니다. 엔트리 심화 프로젝트로 응용 프로그래밍 역량을 키우고, 센서·액추에이터를 제어하는 피지컬 컴퓨팅 실습으로 IoT 개념을 습득하며, 앱인벤터로 실제 배포 가능한 모바일 애플리케이션을 완성합니다.

LV.4
Build
제작 단계
초등 6학년~중등 1학년

인공지능의 학습 원리를 직접 실험하고 제작물로 증명하는 단계입니다. 머신러닝 플랫폼으로 데이터 수집·모델 훈련·성능 평가의 AI 개발 사이클을 경험하고, 피지컬 디바이스와 연동해 실생활 문제를 해결하는 AI 융합 결과물을 설계·구현합니다.

LV.5
Master
통달 단계
중등~

Python을 기반으로 소프트웨어 개발 역량을 종합적으로 완성하는 단계입니다. 정렬·탐색 알고리즘 설계, 데이터 수집·시각화·분석, 머신러닝 모델 구현까지 — 실세계 문제를 스스로 정의하고 해결하는 전 과정을 수행하며, 탐구보고서와 포트폴리오로 결과를 정리해 활용할 수 있습니다.

Graduation

Q-Grand Slam

큐비아 랩의 졸업은 단순한 기간 이수가 아닙니다. 탐구·창작·발표·실행의 다섯 영역에서 균형 있는 성취를 증명해야 완성됩니다.

Q
📜
Qualification
Q1 · 자격 인증

코딩 자격증, SW 관련 공인 시험 등 객관적인 기준으로 실력을 증명하는 단계입니다. 단순 학습을 넘어, 성취를 눈에 보이는 결과로 확인할 수 있습니다. 초등·중등 시기부터 준비한 자격은 고등학교 진학과 이후 진로 선택에서 신뢰도 높은 기반이 됩니다.

Q
🏅
Quest Award
Q2 · 대회 수상

SW·발명·과학·수학 분야의 외부 공인 대회 입상을 통해 실력을 검증하는 단계입니다. 준비 과정에서 문제 해결력과 창의적 사고를 함께 기를 수 있습니다. 이러한 수상 경험은 고등학교에서 중요한 활동 이력으로 이어집니다.

02/05
Q
💻
Quirious Build
Q3 · 디지털 빌드

앱, 웹사이트, 게임, 자동화 프로그램 등 실제로 작동하는 결과물을 직접 기획하고 제작하는 단계입니다. 아이디어를 구현하는 과정을 통해 실전형 문제 해결 능력을 기릅니다. 완성된 결과물은 나만의 포트폴리오가 됩니다.

03/05
Q
📄
Query Report
Q4 · 탐구보고서

문제 제기부터 실험, 데이터 분석, 결론 도출까지의 전 과정을 스스로 수행하는 탐구형 보고서입니다. 단순한 지식 암기를 넘어, 질문하고 검증하는 사고력을 기릅니다. 초등·중등 시기부터 탐구보고서를 꾸준히 작성한 경험은, 고등학교에서 큰 강점으로 이어집니다.

04/05
Q
⚙️
Quantum Make
Q5 · 융합 설계

3D프린팅, 전자회로, 아두이노 등으로 실물 결과물을 직접 설계하고 제작하는 단계입니다. 소프트웨어와 하드웨어를 연결하며 융합적 사고를 확장합니다. 화면 속 코드가 현실에서 작동하는 경험은, 메이커·엔지니어링 계열 진로를 탐색하는 데 가장 강력한 차별점이 됩니다.
